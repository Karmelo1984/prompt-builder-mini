Crear

¿Qué quieres crear?

Sin seleccionar

Destino

¿Dónde lo vas a usar?

Sin seleccionar

Selecciona dónde usarás el artefacto. La compatibilidad depende del tipo que elegiste en el paso anterior.

Perfil

¿Cuál es tu rol o perfil?

Sin seleccionar

Elige un perfil o rol. Cambiar esto reiniciará la plantilla, contexto, restricciones y salida que ya tengas marcadas.

Regla de ahorro

El objetivo no es escribir el prompt más corto, sino evitar varias rondas de correcciones. En conjunto, un prompt estructurado suele gastar menos tokens totales.

Plantilla

Plantilla

Sin seleccionar

Elige la plantilla exacta para tu caso de uso. Cambiarla borra el contexto escrito y las selecciones posteriores.

Información

Rellena el contexto mínimo

Pega solo lo necesario: stack, objetivo, porqué y el input verificable. Si modificas esto después de marcar reglas o formato, esas selecciones se limpian.

Reglas

Define tus restricciones

Define restricciones explícitas para que la IA no se desvíe. Ejemplo: "no reescribas el frontend, solo backend".

Formato

Define el formato de salida

Define el formato de salida esperado: estructura, ejemplos, verificación. Sé específico para evitar literatura decorativa.
